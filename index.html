This an exclusive game only for the Kakima Worshippers

Guess The Song

  • This is a guess the song game.
  • You will be given a picture containing emoji or some text and you have to guess the game from that.
  • You have to write the name of the song in the input field given below and click on play song button.
  • If you don't know the answer just type anything. Unless you type something you can not go further.
  • To move to the next question click on the next button.
